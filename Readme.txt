KTI Template - Avenxo 3.0.0 No Login
=====================================

Description :
Aplikasi ini merupkan template avenxo yang telah di modifikasi sehingga bisa digunakan dengan Codeiginiter,
untuk contoh penggunaan bisa dilihat pada controller application/controller/dashboard.php,
template ini didesain untuk digunakan tanpa melalui login, 
untuk mengakses template Avenxo yang asli bisa dilihat di folder /html_template
